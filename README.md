# Jekyll

Funciona gracias [Jekyll](https://jekyllrb.com/), [Github Pages](https://pages.github.com/) y [SASS](http://sass-lang.com/).

## Run jekyll

```
$ gem install jekyll bundler
$ jekyll new my-awesome-site
$ cd my-awesome-site
$ bundle exec jekyll serve
```
