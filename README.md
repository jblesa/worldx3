# Jekyll

Funciona gracias Jekyll, GithubPages y SASS.

## Run jekyll

```
$ gem install jekyll bundler
~ $ jekyll new my-awesome-site
~ $ cd my-awesome-site
~/my-awesome-site $ bundle exec jekyll serve
```
